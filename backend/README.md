# Employee Management System

## Project Overview
This project is a Full Stack Employee Management System developed using FastAPI and React.js.

The system provides:
- Employee Registration
- JWT Authentication
- Employee CRUD Operations
- Search & Filtering
- Pagination
- Soft Delete
- Swagger API Documentation

## Backend Stack
- FastAPI
- SQLAlchemy
- MySQL
- JWT Authentication

## Frontend Stack
- React.js
- Axios
- Bootstrap

## Development Approach
The project is being developed using a modular and scalable architecture with proper exception handling, clean API responses, and environment-based configuration.
